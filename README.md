# 班表格式轉換工具

餐飲業月排班表格轉換為每日橫向長條圖排班表的網頁工具。

## 專案目標

輸入不同公司/品牌的格狀排班 Excel（員工 × 日期，欄位配置不固定），輸出橫向長條圖排班 Excel（每日一分頁，含 Excel 原生 Gantt 式長條圖），解決格狀班表不利於每日/即時人力控管的問題。

## 計畫架構

- 輸入：Excel 檔案，含「整月外場班表」「整月內場班表」兩張格狀原始輸入表
- 欄位辨識：啟發式猜測（關鍵字比對+型態嗅探）+ 使用者確認畫面，處理主要特徵（姓名/日期/員工編號/職稱）欄位不固定的問題；次要特徵（行銷/活動附註）由使用者圈選後原樣依日期帶入輸出
- 範本儲存：JSON 儲存層存欄位對照範本，使用者上傳時手動選擇範本以重用設定
- 後端：Python（FastAPI/Flask + openpyxl），解析每人每日上下班時間，產生每日分頁 + 原生長條圖
- 前端：上傳頁面 + 欄位對照確認畫面 + 下載頁面
- 輸出：轉換後的 Excel 檔案（每日一分頁，時間軸 8-23 點）
